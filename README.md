# Employee Attendance Management System

This project is a web-based application used to track employee attendance. It allows employees to check in and check out, while the admin can manage employee records and view attendance details.

---

## Features

- Employee check-in / check-out
- Admin dashboard
- Manage employee details
- View attendance records
- Simple and user-friendly interface

---

## Technologies Used

- React JS, Tailwind CSS, JavaScript
- Node JS
- Express
-MongoDB
